Exemplo de justify-content

O que é juntify-content É a propriedade que determina como os itens em um container flex são distribuidos ao lingo do eixo principal. Se o eixo principal for horizontal (padrão para flex-direction: row ), os itens são alinhados horizontalmente de diferentes formas, dependendo do valor escolhido.

justify-content: flex-start

Descrição Os itens são colocados do container(lado esquerdo, no caso de direção em linha.)

justify-content: flex-end

Descrição Alinha os itens ao final do contêiner (parte inferior, no caso de um eixo horizontal principal)

justify-content: flex-center

Descrição Alinha os itens ao centro do eixo vertical

justify-content: flex-baseline

Descrição Alinha os itens pela linha de base do texto (útil quando os itens têm tamanhos de fontes diferentes)

justify-content: flex-stretch

Descrição Os itens se esticam para preencher todos o espaço disponivel no eixo transversal(desde que nenhum valor fixo de altura seja definido para o item)
