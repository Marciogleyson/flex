O que é clip-path

O clip-path é uma propriedade que permite criar formas complexas cortando elementos de acordo com o caminho especifico. É util para criar layouts e efeitos visuais interessantes.

Como funciona o clip-path

o clip-path: define uma área visivel de um elemento, escondendo o que esta fora dessa área. Existe várias formas de usar o clip-pathcomo:

  • polygon(): Define uma forma usando uma lista de coordenadas que forman um poligono.
  • circle(): Define uma forma circular.
  • ellipse: Define uma forma elíptica.
  • inset(): Define um retângulo interno com uma margem opcionl.
